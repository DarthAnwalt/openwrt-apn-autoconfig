# Contributing

Contributions must be original work that the contributor has the right to
submit, or must identify every third-party source and its license. Do not copy
code, data, documentation, logos or other material from an unverified source.

Provider changes must include reproducible provenance: an operator's official
public documentation or a dated real-network test description. Do not submit
personal subscriber credentials, ICCIDs, IMSIs, phone numbers, private keys or
router backups.

This project uses the Developer Certificate of Origin 1.1. Add a sign-off to
each commit with `git commit -s`. By signing off, you certify that you wrote the
contribution or otherwise have the right to submit it under the project's
applicable license, and that the contribution may be distributed as part of
this project. The full DCO is available at https://developercertificate.org/.

Original implementation and local provider overrides are accepted under the
MIT License. Imported provider data remains under the upstream terms described
in `LICENSING.md`. A contribution must not relabel upstream data as MIT.
